## 三条产品线，先搞清楚分别是什么

DMIT 目前运营洛杉矶（LAX）、香港（HKG）、日本东京（TYO）三个数据中心，每个机房都分三档线路：

**Premium（Pro 系列）**：旗舰档。电信联通去程 CN2 GIA，移动去程 CMI，三网回程 CN2 GIA 双向优化。晚高峰最稳，延迟最低，价格也最贵。主要面向对网络质量有硬性要求的用户——建站服务国内用户、搭梯子、跑对延迟敏感的业务。

**Eyeball（EB 系列）**：中档。洛杉矶版走电信联通 AS9929 + 移动 CMIN2；香港版三网 CMI。比 Pro 便宜，流量配额比同价 Pro 套餐更大，联通和移动用户的体验特别好，电信用户略差一点。性价比最高的选择。

**Tier 1（T1 系列）**：经济档。国际线路，没有针对中国大陆的专属优化。价格最低，流量配额最大。流量超出后不停机，自动限速继续用——WEE 套餐超量后限速到 50Mbps，对个人用户来说基本不会突然断网。适合面向海外用户的业务，或者就是想要一个便宜稳定的美国节点。

## 不同需求场景下怎么选

**只是练手、跑脚本、部署个人博客，不太在意国内访问速度** → 洛杉矶 T1 系列，年付最低 $36.9 左右，够了。不要过度买。

**主要服务国内用户或者需要稳定翻墙节点，电信用户为主** → 洛杉矶 LAX.Pro，三网 CN2 GIA，晚高峰不会掉速。AN4 平台 TINY 套餐 $88.88/年，算下来每月不到 $7.5，这个价位的 CN2 GIA 正经不多。

**联通或移动用户，想要优化线路但不想付 Pro 的价钱** → 洛杉矶 LAX.EB 系列，9929 + CMIN2 回程，对联通移动的体验甚至比 Pro 更均衡，而且流量比 Pro 同价套餐多不少。

**需要极低延迟，主要服务中国大陆用户** → 香港 Pro 系列，延迟基本在 10-50ms 之间，这是物理距离的优势。就是贵，TINY 套餐 $39.90/月，不适合个人小项目。

**大流量业务，带宽比速度重要** → 洛杉矶 T1 大流量（AN5）系列，单套餐流量从 5TB 起，10Gbps 带宽，性价比在大流量场景里很突出。

## FAQ

**Q：DMIT 适合搭建梯子吗？**

适合，但要看你用哪个套餐。电信用户首选 LAX.Pro 系列，CN2 GIA 三网回程，晚高峰延迟稳定。联通移动用户选 LAX.EB，9929 + CMIN2 回程，性价比更高。如果只是个人用，LAX.Pro TINY 或 LAX.EB TINY 的年付方案都够了。

**Q：DMIT 的便宜VPS里最便宜的是哪款？**

洛杉矶 T1 系列的 WEE 套餐年付约 $36.9，折算下来每月 $3 出头，是 DMIT 里价格最低的入门档。没有大陆线路优化，但硬件稳定，流量配给足，超量限速不断网，拿来跑面向海外的小项目或者测试环境很合适。

**Q：DMIT 和搬瓦工比哪个好？**

两家都做 CN2 GIA 线路，定位接近。DMIT 在同配置下价格通常便宜一些，硬件也比搬瓦工部分老套餐新。搬瓦工的优势是体系更成熟、用的人多、出问题更容易找到参考。两家都有 3 天无理由退款政策，实在纠结就分别测一下。

**Q：DMIT 支持哪些付款方式？**

支付宝、微信支付、PayPal、信用卡，对国内用户来说付款没什么障碍。

**Q：香港 VPS 一定比洛杉矶好吗？**

延迟方面是的——香港到大陆主要城市 50ms 以内，洛杉矶 Pro 系列大概 150-180ms，差距明显。但香港价格比洛杉矶贵得多，而且库存经常缺货。如果业务对延迟不是极度敏感，洛杉矶 Pro 已经够用。如果你服务的是实时交互类场景（直播、游戏服务器、企业内网），香港才真的值得多出这部分钱。
